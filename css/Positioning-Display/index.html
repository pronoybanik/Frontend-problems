4. Overflow Property

Overflow: Visible (default)

This content overflows the container and is visible outside the boundaries. You can see it extending beyond the box.

Overflow: Hidden

This content overflows but is clipped and hidden. You cannot see the content that extends beyond the container boundaries.

Overflow: Scroll

This content overflows but you can scroll to see it all. Scrollbars appear even if content fits. Try scrolling in this box!

Overflow: Auto

This content overflows and scrollbars appear automatically only when needed. This is the most commonly used overflow value.
